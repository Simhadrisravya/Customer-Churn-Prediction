# Customer Churn Prediction

## 📌 Project Overview
Predict whether a customer will churn using machine learning.

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📊 Dataset
Customer churn dataset with telecom customer details.

## 📈 Results
- Accuracy: ~80%
- Random Forest performed best

## 🛠️ Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Key Insights
- Month-to-month customers churn more
- High charges increase churn
- Low tenure customers churn more
